# AST_Final_Project

## Introduction :
This project is about the medical appointment dataset. The dataset and the python file is attached in this project folder. This appointment dataset contains 14 features which describes the appointment details like, Patients_Id, Appointmnet_Id, Date of appointmnet, Age, Gender, etc. 

Our main objective is to obtain inferences through exploratory data analysis and implementing Ensemble method - Random forest and Dimensional Reduction method - PCA (Principal Component Analysis).
